Student Management System
Build Instructions

Technology:
- Java 21
- Spring Boot 3.x
- Maven
- MySQL

Prerequisites:
1. Install Java JDK 21 or later.
2. Install Apache Maven 3.9 or later.
3. Install MySQL Server.
4. Create a database named studentdb.

Build Command:
mvn clean install

Run Command:
mvn spring-boot:run

Alternative Run Command:
java -jar target/student-system-0.0.1-SNAPSHOT.jar

Application URL:
http://localhost:8080

Database Configuration:
spring.datasource.url=jdbc:mysql://localhost:3306/studentdb
spring.datasource.username=root
spring.datasource.password=your_password

Cloud Deployment:
For Railway or other cloud platforms, configure these environment variables:
SPRING_DATASOURCE_URL
SPRING_DATASOURCE_USERNAME
SPRING_DATASOURCE_PASSWORD
